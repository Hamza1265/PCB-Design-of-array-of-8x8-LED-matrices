Introduction Of Project

The LED Matrix Display project concerns the construction of a two-dimensional arrangement of LEDs in a rectangular arrangement for the purpose of displaying English alphabets (upper case) and decimal numbers. The characters to be displayed are entered using a computer. Therefore, in this project, the computer functions as an input device and the LED matrix display functions as an output device. A computer program should also be included in the project to create a user interface environment for entering characters and numbers to be displayed.
